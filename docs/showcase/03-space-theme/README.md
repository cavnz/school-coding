# Space Theme Example

This example demonstrates how to create a **Space/Star Wars inspired theme** using mainly colors and small drawing additions to the starter template.

## Key Theming Changes

### Colors
- **Background**: Deep space black-blue (`rgb(10, 10, 30)`)
- **Player**: Bright cyan spaceship (`rgb(100, 200, 255)`)
- **Platforms**: Metallic gray (`rgb(80, 80, 90)`)
- **Coins**: Cyan energy crystals (`rgb(0, 255, 200)`)
- **Dangerous platforms**: Red laser beams (`rgb(255, 50, 50)`)
- **Score**: Bright yellow sci-fi text (`rgb(255, 255, 100)`)

### Drawing Enhancements
- **Player**: Triangle spaceship with cockpit window and thruster flames
- **Coins**: Diamond-shaped energy crystals with pulsing glow effect
- **Background**: 100 twinkling stars
- **Platforms**: Metallic highlight edge for 3D effect
- **Dangerous platforms**: Animated laser beams with scan lines and glow

### Physics
- Lower gravity (0.3) for "space feel"

### Terminology
- Coins → "Energy Crystals"
- Score → "ENERGY"
- Dangerous platforms → "Laser Beams"

## Teaching Value

Shows students how:
- Color palette alone creates strong atmosphere
- Simple shape changes (rectangle → triangle) transform meaning
- Canvas glow effects (`shadowBlur`, `shadowColor`) add polish
- Background animations (stars) add depth
- Terminology changes strengthen theme
