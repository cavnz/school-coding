# K-Pop Theme Example

This example demonstrates how to create a **K-pop inspired theme** with vibrant colors, sparkles, and cute character design from the starter template.

## Key Theming Changes

### Colors
- **Background**: Pink-to-purple gradient (`rgb(255, 105, 180)` → `rgb(138, 43, 226)`)
- **Player**: Hot pink with cute face (`rgb(255, 105, 180)`)
- **Platforms**: Purple with gradient (`rgb(138, 43, 226)`)
- **Coins**: Light pink hearts (`rgb(255, 192, 203)`)
- **Dangerous platforms**: Yellow lightning bolts (`rgb(255, 255, 0)`)
- **Score**: White text with gold glow and heart decorations
- **Accent**: Gold (`rgb(255, 215, 0)`) for highlights

### Drawing Enhancements
- **Player**: Rounded character with kawaii face (eyes, smile) and headphones
- **Coins**: Heart shapes using bezier curves with pulsing animation
- **Background**: 50 colorful falling sparkles with rainbow hues
- **Platforms**: Gradient fill with sparkle highlight
- **Dangerous platforms**: Zigzag lightning bolts with pulsing glow
- **Score**: Pulsing animation with heart decorations
- **Particles**: Star shapes instead of circles

### CSS Effects
- Animated glowing title with `@keyframes`
- Gradient background on page
- Multiple text shadows for sparkle effect
- Rounded canvas corners
- Gold border with dual-color glow

### Terminology
- Coins → "Hearts"
- Score → Display with heart symbols `♥`
- Dangerous platforms → "Lightning Bolts"

### Sound Design
- Sine wave oscillator for softer, cuter sounds
- Higher frequencies for cheerful feel

## Teaching Value

Shows students how:
- Multiple gradients create depth and vibrancy
- Bezier curves create complex shapes (hearts)
- CSS animations enhance the whole page experience
- Character design with simple shapes creates personality
- Particle shapes can be customized (stars vs circles)
- Sound wave types affect emotional tone
- Color harmony (pink/purple/gold) creates cohesive theme
