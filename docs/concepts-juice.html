What is "Juice"?

"Juice" is the term game developers use for all the visual and audio effects that make a game feel satisfying and fun. It's the particles, screen shake, sound effects, and animations that happen when you do something!

This section teaches you how to add juice by using functions!

üéØ Functions: Instructions to the Computer

What is a Function?

A function is like a recipe or a set of instructions that you give to the computer. You give the recipe a name, and whenever you "call" that name, the computer follows those instructions.

Think of it like teaching a robot a dance move:

  • Define the function: You teach the robot: "When I say 'spin', do a 360¬∞ turn"
  • Call the function: Later you shout "spin!" and the robot does it
  • Reuse it: You can say "spin" as many times as you want, and the robot remembers what to do

Writing a Function

Here's how you create a function in JavaScript:

// Define a function called "sayHello"
function sayHello() {
  console.log('Hello, world!');
  console.log('Welcome to coding!');
}

// Now you can "call" the function - run those instructions
sayHello();  // This will print both messages

// Call it again!
sayHello();  // Prints the messages again

Functions Can Accept Information

Functions can receive parameters (inputs) to work with:

// Function that takes a name as input
function greet(name) {
  console.log('Hello, ' + name + '!');
}

// Call it with different names
greet('Alice');   // Prints: Hello, Alice!
greet('Bob');     // Prints: Hello, Bob!

// Functions can take multiple parameters
function addNumbers(a, b) {
  const sum = a + b;
  console.log(sum);
}

addNumbers(5, 3);   // Prints: 8
addNumbers(10, 20); // Prints: 30

Chain Reactions: Functions Calling Functions

This is where it gets really cool! One function can call other functions, creating chain reactions:

// When a coin is collected, this function runs:
function onCoinCollected(coin) {
  // This ONE function calls multiple other functions!
  spawnParticles(coin.x, coin.y, '#FFD700', 20);  // Makes particles
  screenShake(5);                              // Shakes screen
  beep(660, 100, 0.3);                       // Makes sound
}

// The game system checks for coin collision and calls this:
if (playerTouchingCoin) {
  collectCoin(coin);  // This system function...
}

function collectCoin(coin) {
  score += POINTS_PER_COIN;            // Update score
  coins.splice(coins.indexOf(coin), 1); // Remove coin
  onCoinCollected(coin);               // ...calls YOUR effects function!
}

// ONE coin collection triggers a CHAIN REACTION:
// 1. Player touches coin (collision detected)
// 2. collectCoin runs (system handles score & removal)
// 3. onCoinCollected runs (YOUR effects!)
// 4. spawnParticles runs
// 5. screenShake runs
// 6. beep runs
// All from one event!

Why Functions Are Amazing

  • Reusable: Write the code once, use it many times
  • Organized: Each function has a clear purpose (jump, draw, shake, etc.)
  • Easier to read: onJump() is clearer than copy-pasting 10 lines of code
  • Easy to change: Update one function and it changes everywhere it's used
  • Chain reactions: Create complex effects by combining simple functions

onJump()

function onJump()

When it's called: When the player presses space and jumps off the ground

function onJump() {

  screenShake(3);

function onLand()

When it's called: When the player lands on a platform after falling

üé® The drawXYZ Functions: Simple Drawing Tools

The game includes helpful drawing functions that make it easy to draw different shapes! You can use these in the drawPlayer(), drawCoins(), or other drawing functions to customize how things look.

The Drawing Functions

drawRect(x, y, width, height, color)

drawRect(x, y, width, height, color)

What it does: Draws a rectangle

Parameters:

  • x, y - Top-left corner position
  • width, height - Size of the rectangle
  • color - Color (try 'red', 'rgb(255, 0, 0)', or '#FF0000')
// Draw a blue rectangle for the player
drawRect(player.x, player.y, player.width, player.height, PLAYER_COLOR);

// Draw a red hat on the player
drawRect(player.x, player.y - 10, player.width, 5, 'red');

drawCircle(x, y, size, color)

drawCircle(x, y, size, color)

What it does: Draws a circle

Parameters:

  • x, y - Center position of the circle
  • size - Diameter of the circle
  • color - Color of the circle
// Draw white eyes on the player
drawCircle(player.x + 10, player.y + 10, 8, 'white');  // Left eye
drawCircle(player.x + 20, player.y + 10, 8, 'white');  // Right eye

// Draw circular particles
drawCircle(particle.x, particle.y, 5, particle.color);

drawTriangle(x, y, size, direction, color)

drawTriangle(x, y, size, direction, color)

What it does: Draws a triangle pointing in any direction

Parameters:

  • x, y - Center position of the triangle
  • size - How big the triangle is
  • direction - Which way it points: 'up', 'down', 'left', or 'right'
  • color - Color of the triangle
// Draw a party hat on the player
drawTriangle(player.x + player.width / 2, player.y - 10, 20, 'up', 'red');

// Draw spikes pointing down on platforms
drawTriangle(platform.x + 50, platform.y, 15, 'down', 'red');

drawText(x, y, text, size, color)

drawText(x, y, text, size, color)

What it does: Draws text or emojis (automatically centered)

Parameters:

  • x, y - Center position of the text
  • text - What to draw (can be text or emojis!)
  • size - Font size in pixels
  • color - Color of the text
// Draw an emoji character as the player
drawText(player.x + player.width / 2, player.y + player.height / 2, 'üòÄ', 30, 'white');

// Draw star emojis as coins
drawText(coin.x, coin.y, '‚≠ê', COIN_SIZE, 'yellow');

// Draw a victory message
drawText(canvas.width / 2, 100, 'YOU WIN! üéâ', 48, 'yellow');

Example: Custom Player Character

Here's how you could customize the drawPlayer() function:

function drawPlayer() {
  const centerX = player.x + player.width / 2;
  const centerY = player.y + player.height / 2;

  // Option 1: Simple emoji character
  drawText(centerX, centerY, 'üòä', player.width, 'white');

  // Option 2: Build a character with shapes
  drawCircle(centerX, centerY, player.width, PLAYER_COLOR);  // Body
  drawCircle(centerX - 8, centerY - 5, 6, 'white');     // Left eye
  drawCircle(centerX + 8, centerY - 5, 6, 'white');     // Right eye
  drawTriangle(centerX, centerY - player.width / 2 - 10, 20, 'up', 'red');  // Hat
}

üí• Effect Functions: The Juice Makers

These are helper functions you can call from the onXYZ() functions to add effects to your game!

spawnParticles(x, y, color, count)

spawnParticles(x, y, color, count)

What it does: Creates colorful particles that shoot out and fall down

Parameters:

  • x, y - Where to spawn the particles
  • color - Color of the particles
  • count - How many particles to spawn
// Gold explosion when collecting a coin
spawnParticles(coin.x, coin.y, '#FFD700', 20);

// Rainbow particles
spawnParticles(player.x, player.y, 'rgb(255, 0, 255)', 10);

screenShake(intensity)

screenShake(intensity)

What it does: Makes the screen shake briefly

Parameters:

  • intensity - How hard to shake (try 3 for small, 10 for big)
// Small shake when jumping
screenShake(3);

// Big shake when landing
screenShake(10);

beep(frequency, duration, volume)

beep(frequency, duration, volume)

What it does: Plays a simple beep sound

Parameters:

  • frequency - How high/low the sound is (220-880 works well)
  • duration - How long in milliseconds (100 = short, 500 = long)
  • volume - How loud (0.1 = quiet, 0.5 = loud)
// Jump sound (mid pitch, short, quiet)
beep(440, 100, 0.2);

// Coin collection (high pitch, short, medium volume)
beep(660, 100, 0.3);

// Landing thud (low pitch, short, medium volume)
beep(220, 80, 0.3);

Combining Effects

The magic happens when you combine multiple effects together!

function onCoinCollected(coin) {
  // Create an epic coin collection with all three effects!
  spawnParticles(coin.x, coin.y, '#FFD700', 20);  // Gold explosion
  screenShake(5);                               // Screen shake
  beep(660, 100, 0.3);                        // "Ding!" sound
}
